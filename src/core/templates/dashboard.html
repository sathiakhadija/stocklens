ABC-XYZ Inventory Classification

ABC = revenue contribution over last 90 days: A = top 80% · B = next 15% · C = bottom 5%.
XYZ = demand variability (CV): X = stable (CV<0.5) · Y = moderate · Z = erratic (CV≥1.0).
Combined class determines review frequency and ordering strategy.
